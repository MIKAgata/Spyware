# ScreenStreamer - Professional Screen Streaming Reverse Shell
Image not available
High-performance, stealthy screen capture & streaming untuk penetration testing dan authorized security assessments

## Fitur Utama
| Feature | Description |
| --- | --- |
| Ultra-Fast Streaming | 15+ FPS dengan <5MB/s bandwidth |
| Auto-Reconnect | Koneksi otomatis reconnect |
| Smart Resizing | Auto-resize ke 1280px max |
| Region Capture | Streaming area layar spesifik |
| Stealth Mode | Minimal CPU/RAM, no GUI flicker |
| Live Metrics | FPS, RAM monitoring real-time |
| Configurable | .env support, full customization |
| Graceful Shutdown | SIGINT/SIGTERM handling |
